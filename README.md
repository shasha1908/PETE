# PETE
Python based Assistant
Libraries used:
1) pyttsx3
2) datetime
3) speech_recognition
4) wikipedia
5) webbrowser
6) os
7) smtplib
8) googlesearch(in the next update)

Features in the first version:

1) Wish the user
2) Wikipedia search
3) Open YouTube
4) Open stackoverflow
5) Open facebook
6) play music
7) Time
8) open code file
9) open files
10) send email(but only if you have a business email-id
